# Auto-Configurator Utilities

## Overview

Intelligent project setup and configuration utilities.

## Features

- Automatic dependency detection
- WebDriver setup (Chrome, Firefox, Edge, Safari)
- Configuration file generation
- Environment validation
- Pre-commit hook setup

## Usage

```bash
python auto_configure.py
```

The script will:
1. Detect installed browsers
2. Download appropriate WebDriver
3. Generate configuration files
4. Set up pre-commit hooks
5. Validate environment
